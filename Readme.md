# Johnson

This is a java implementation of the Johnson's simple cycles algorithm. 

## Requirements

 - Java Runtime Version 11
 - Maven 3.1
 - junit 5.6
 - glassfish 2.28

 These requirements are also listed inside the `pom.xml` since this is a maven project.
